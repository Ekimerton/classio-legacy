What is classio?

Classio is an app that helps you pick classes. All you do is pick your university and enter your class codes!

How do I get in touch if I have an idea for a feature or if I find a bug?

You can leave anonymous feedback here.

What does the percentage in my results mean?

The percent is a score, calculated using the factors you care about. (By default those factors are lunches missed, dinners missed and time wasted between classes) You can change how much you care about each factor in Advanced Settings.

What "Early/Midday/Late/Mixed" mean?

Early means classes start between 11:30 on most days of the week. Midday means classes start between 11:30 and 4:30. Late means classes start after 4:30. Mixed means most days of the week have a starting time and an ending time far apart from each other.

What does it mean if I get 0 results back?

0 results could mean a lot of things. Firstly, your entry might be formatted incorrectly, so the app might not have found the class times. It could also be that there are no combinations in which the classes you entered work together.

I got a heroku error, what does this mean?

A heroku error occurs if your results take more than 30 seconds to be generated (eg. if you enter too many classes). Sadly, the only solution to this is running classio locally, which you can do by downloading it here. If you'd like a step-by-step on how to run classio locally, refer to the repository's README.
